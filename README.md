Project Overview

This project is a Power BI dashboard that analyzes employee performance, attendance, and salary data to provide insights for better decision-making.

Tools Used
-Microsoft Power BI
-Excel (Data Source)
-Data Cleaning & Visualization

Key Features
-Employee performance tracking
-Attendance analysis
-Salary insights
-Interactive dashboards with filters

Insights Generated
-Identified top-performing employees
-Analyzed attendance patterns
-Compared salary distribution across departments

 File Included
  employee_report.pbix (Power BI Dashboard)

 How to Use
  Download the .pbix file
  Open it in Power BI Desktop
  Explore dashboards using filters and visuals
